Research

Research Interests

  • Code Generation and Optimization
    • Program transformations using the polyhedral model. Related publications: CC 2018, ACM TACO 2019, MICRO 2020, ACM TOCS 2024.
    • Code generation for high-performance architectures. Related publications: ICS 2019, ICPP 2022, PPoPP 2024.
  • Deep Learning Systems
    • Frameworks for deploying deep learning workloads on distributed architectures. Related publications: arXiv 2021.
    • Graph compilers that study the fusion of graph nodes. Related publications: MLSys 2022, OSDI 2023.
    • Operator compilers that investigate automatic loop transformations and memory managements for tensor operations. Related publications: PLDI 2021, PACT 2022, MLSys 2023, OSDI 2024.
  • Numerical Program Analysis
    • Floating-point error detection. Related publications: ASE 2023.
    • Repairing numerical programs through expression rewriting: ISSTA 2024.
